saya

DTV backend service with powerful features such as...

  • Streaming comments from niconico
  • Fetching anime information from Annict

Endpoint Description Parameters
WS
/comment/stream/{target}
Streams live comments.
Response: JSON over WebSocket
Error: 404 if {target} not found.
{target}
niconico Jikkyo ID (like jk1),
or DTV service id (like 211).
GET
/comment/stats/{target}
Returns comment statistics.
Response: JSON
Error: 404 if {target} not found.
{target}
Same as /comment/stream/{target}
